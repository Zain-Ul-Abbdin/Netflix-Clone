# Netflix Clone

A responsive Netflix clone built with HTML, CSS, and JavaScript.

## Features

- Responsive design that works on all devices
- Fixed navigation bar with scroll effects
- Hero section with featured content
- Multiple content rows with horizontal scrolling
- Hover effects on movie posters
- Footer with links and language selector
- Basic interactivity with JavaScript

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (ES6+)
- Font Awesome for icons

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser or run a local server
3. For local server: `python3 -m http.server 8000` and visit `http://localhost:8000`
